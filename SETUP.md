# ⚙ CartaOS Setup Guide

---

## 🖥 Requirements
- Python 3.10+
- Node.js 18+
- Rust
- Tesseract OCR
- poppler-utils

---

## 🚀 Install
```bash
git clone https://github.com/CartaOS/CartaOS.git
cd CartaOS
python3 backend/cartaos/install_dev_env.py
````

---

## ▶ Run CLI

```bash
python3 backend/cartaos/cli.py
